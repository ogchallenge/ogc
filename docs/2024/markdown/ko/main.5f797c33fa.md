## 대회 개요

**2024 최적화 그랜드 챌린지(Optimization Grand Challenge 2024)** 는 실세계 배달 최적화 문제를 해결하는 알고리즘 경진대회입니다.

### 주요 목표

본 대회는 배달 서비스의 핵심 난제인 **묶음 배달(Bundle Delivery)** 최적화를 주제로 합니다. 참가자들은 다음을 달성하는 알고리즘을 개발해야 합니다:

- 🚴 **효율적인 주문 묶음 배달** - 여러 주문을 하나의 배달원에게 할당
- 🗺️ **최적 배달 경로 생성** - 시간과 거리를 최소화하는 경로 계획
- ⏰ **데드라인 준수** - 모든 주문의 배달 약속 시간 보장
- 💰 **비용 최소화** - 배달원 수와 이동 거리 최적화


### 배달 최적화 문제란?

음식 배달 앱을 통해 주문이 들어오면, 여러 주문을 묶어서 한 명의 배달원에게 할당하고 최적의 경로를 계획해야 합니다. 이때 다음 제약 조건을 만족해야 합니다:

- 각 주문은 준비 시간(ready time) 이후에 픽업 가능
- 모든 주문은 데드라인(deadline) 이전에 배달 완료
- 배달원의 용량(capacity) 제한 준수
- 이동 시간과 배달 비용 최소화
